Claudometer
Chrome Extension

Privacy Policy

Last updated: May 2026

The short version: Claudometer collects nothing, stores nothing externally, and sends no data anywhere. Everything stays in your browser.

What Claudometer does

Claudometer is a Chrome extension that reads your Claude.ai session usage data and displays it as a colour-coded icon in your browser toolbar. It tells you how much of your 5-hour session, weekly allowance, and (where applicable) weekly Opus allowance you have used.

Data we collect

We collect no data whatsoever. Claudometer does not:

  • Collect, store, or transmit any personal information
  • Read your Claude conversations or messages
  • Track your browsing activity
  • Use analytics or crash reporting tools
  • Connect to any third-party servers
  • Require an account or sign-up

How it accesses Claude data

Claudometer reads your Claude.ai usage statistics by calling the same internal API endpoint that Claude's own settings page uses (claude.ai/api/organizations/{id}/usage). Your existing browser login session authenticates this request automatically — the extension code never reads, stores, or transmits your session cookie. The browser handles authentication the same way it does for any normal webpage request.

What is stored locally

The only data stored on your device (using Chrome's local storage API) is:

  • Your Claude organisation ID — a non-personal identifier used to avoid repeated API lookups, cached for 24 hours
  • The last fetched usage percentages — used to display data in the popup without an additional network request

This data never leaves your device and can be cleared at any time via the "Reset org cache" button in the extension popup, or by uninstalling the extension.

Permissions

Claudometer requests only two browser permissions:

  • storage — to save the organisation ID and last-known usage data locally on your device
  • alarms — to trigger a usage check every 60 seconds using Chrome's built-in timer system

The extension also declares host access to claude.ai only, so it can make API requests to that domain. It has no access to any other website.

Third parties

Claudometer has no backend server, no analytics service, no advertising, and no third-party integrations of any kind. No data is shared with or sold to any third party.

Children's privacy

Claudometer does not knowingly collect any information from anyone, including children under the age of 13.

Changes to this policy

If this policy changes, the updated version will be posted here with a revised date. Given the nature of the extension, significant changes are unlikely.

Contact

If you have any questions about this privacy policy, you can reach us through the Chrome Web Store listing page.
